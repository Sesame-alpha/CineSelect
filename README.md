# CineSelect

Responsive cinema seat selector built with HTML, CSS, and JavaScript for the Coursera "Create a responsive seat selector UI" project.

## Features
- Visual seat map (8 rows × 10 seats) with center aisle
- Premium row pricing (rows G & H) vs standard
- Seat states: available, selected, occupied
- Live booking summary with seat count and total price
- Booking persists via localStorage (seats stay taken after refresh)
- Fully responsive, mobile-friendly grid
